Available Endpoints:

GET /hello

Returns "Hello Express JS" as plain text.

Example: GET http://localhost:3000/hello

GET /user

Returns user data using query parameters for firstname and lastname.

Defaults to "Pritesh" and "Patel" if parameters are not provided.

Example: GET http://localhost:3000/user?firstname=John&lastname=Doe

POST /user/:firstname/:lastname

Returns user data using path parameters for firstname and lastname.

Example: POST http://localhost:3000/user/John/Doe

POST /users

Returns an array of users. Accepts body parameter for array of users with firstname and lastname.

Returns default users if no body is provided.

Example: POST http://localhost:3000/users

Body example: {"users": [{"firstname": "John", "lastname": "Smith"}]}

How to Test:

  1. Start the server: npm start or node server.js
  2. Use Postman or any API testing tool to test the endpoints
  3. This instruction page is accessible at: http://localhost:3000/instruction.html
